• factlink-core: Runs the factlink.com website, and the discussion page that runs in an iframe.
  • js-library: The Javascript annotation library that is included on websites, browser extensions, and in the proxy.
  • web-proxy: Proxies arbitrary pages, and injects a script tag to the js-library. Runs on fct.li.
  • browser-extensions: Extensions for Chrome, Firefox, and Safari. Just inserts script tag to js-library.
  • wordpress-plugin: Loads js-library on a Wordpress blog.
  • pivotx-plugin: Loads js-library on a PivotX site.
  • url_normalizer: Gem for converting an arbitrary URL to a canonical URL.
  • factlink_blacklist: Regexes that matches URLs on which Factlink should be disabled for technical or privacy reasons.
